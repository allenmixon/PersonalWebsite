Projects:

Timber Count - App designed to help smaller sawmills manage their data at an affordable price. Larger companies pay for expensive software. This helps them stay competitive. Data input comes directly from workers on the cutting floor. Android tables are used for data entry. Everything is kept up with on a remote Linux server and live updates are displayed for guys back in the office. First phase of this project is complete and is currently being tested at Greentree Lumber.

MixGeo - Web application designed to facilitate the permitting phase of seismic surveys. Incorporates open source GIS technology such as PostGIS and GeoServer. Everything is kept up with on a remote Linux server using Postgresql. Easy user interface so that anyone can update data and the changes are disseminated instantaneously. The application was written using Python's Flask web-framework.
